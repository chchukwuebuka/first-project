CHALENGE

QR code component

A perfect first challenge if you're new to HTML and CSS. The cardlayout doesnt shift, so its ideal if you haven't learned about building responsive layout yet

Brief

your challengeis to build this QR code component and get it lookin as close to the design as possible.
You can use any tools you like to helpyoucomplete the challenge. Soif you've got something you'd like to practice, feel free to give it a go.
Download the starter code and go through the README.md file.This will provide further details about the project. The style-guide.md fiesis whee you'll find colors,fonts,etc.
Want some support on the challenge? join our community and ask questions in the help channel.

Assets provided

sketch and figma design file access -Unlock with Pro
JPEG design file for mobile & desktop layoit
Style guid for fonts, colors, etc.
Optimized image assets
README file to help you get started

Ideas to test yourself

  1. Write your style using a pre-processor, such as sass, less or stylus
  2. Train your eye for detail by getting your solution as close to the design as you can
  3. Try estimating the time it will take fo you to build the project. then see it the time take matches up to your estimated. Project estimations are a skill that is ofton overlooked, but is important fo professional developers
